# 项目简介

## 核心功能
- 基于YOLOv8的遥感太阳能板实例分割检测
- 支持图片、视频及摄像头进行检测
- 支持选择模型和自定义置信度、IOU阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 提供35000张包含'solar panel'类别的数据集
- 包含训练好的模型及训练结果

## 适用场景
- 遥感图像分析
- 太阳能板检测与识别
- 环境监测与资源普查
- 大规模图像批量处理

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整程序文件、UI界面源文件、图标及数据集
- 配置成功环境可直接使用
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
